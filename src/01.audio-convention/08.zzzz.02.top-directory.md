# 顶级目录

顶级目录负责将音乐资源按**类型**和**名称**分隔为各个大类。

## 目录名

顶级目录的目录名格式如下：

```
[{type}] {name}
```

其中 `type` 取[目录类型][directory-type]中的标识符，`name` 则为对应名称。

[directory-type]: ./08.zzzz.01.directory-type.md
