# annil-rs

`annil-rs` 是最简单的音频仓库实现之一。其本身只实现了对 `HS256` 签名的 `JWT` 的校验，只要校验通过就允许用户访问资源。

## 配置文件

```toml
[server]
# 音频仓库名称
name = "My Server"
# 仓库监听的地址
listen = "localhost:3614"
# 仓库签名 JWT 的密钥
hmac-key = "a hmac key"

[[backends]]
# 音频后端名称
name = "File Backend"
# 是否启用该音频后端
enable = true
# 音频后端类型，此处为 file
type = "file"
# FileBackend 的参数，根目录
root = "/path/to/backend"
# 是否为严格文件后端
strict = false
```
