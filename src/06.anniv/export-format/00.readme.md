# 信息导出格式

`Annil` 提供了分享 `Token` 的创建功能，但这只是资源分享的基石。

对于音频分享而言，我们需要的不只是音频文件的集合，还需要包含顺序、元数据等在内的一系列信息。而这一切需求汇总到一起，信息导出格式便应运而生。

## 播放列表导出

  "name": "Example", // 播放列表名称
  "description": "Example description", // 播放列表简介
