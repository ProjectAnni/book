# 搜索

## Endpoint

`POST /api/search`

## 请求参数

| 参数名    | 类型     | 说明         |
| --------- | -------- | ------------ |
| `album`   | `bool`   | 是否搜索专辑 |
| `track`   | `bool`   | 是否搜索音轨 |
| `keyword` | `string` | 搜索关键字   |

## 返回

返回 `SearchResult` 。

```ts
interface SerachResult {
  albums: AlbumInfo[];
  tracks: TrackInfoWithAlbum[];
  playlists: Playlist[];
}
```

## 错误列表

| 错误代码 | 详情       |
| -------- | ---------- |
| `902002` | 用户未登录 |
