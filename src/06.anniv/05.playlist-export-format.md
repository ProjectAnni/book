# 播放列表导出格式

为分享需要，此处定义 `Anniv` 的播放列表导出格式。

```json
{
  // 播放列表名称
  "name": "22/7",
  // 播放列表简介
  "description": "22/7 所有非 instrumental 歌曲列表",
  // 按顺序存放播放列表中的所有歌曲
  "songs": [
    {
      "catalog": "SRCL-9520",
      "tracks": [1, 2, 3]
    },
    {
      "catalog": "SRXX-2666",
      "tracks": [1, 2, 3, 4, 5],
    },
    {
      "catalog": "SRXX-2851",
      "tracks": [1, 2, 3, 4, 5],
    },
    {
      "catalog": "SRXX-3211",
      "tracks": [1, 2, 3, 4, 5],
    },
    {
      "catalog": "SRXX-3425",
      "tracks": [1, 2, 3, 4, 5],
    },
    {
      "catalog": "SRXX-3580",
      "tracks": [1, 2, 3, 4, 5],
    },
    {
      "catalog": "SRXX-3853",
      "tracks": [1, 2, 3, 4, 5],
    }
  ],
  // 存放播放列表中的元数据
  // 如果导出的目的并非分享，可以省略
  // 此处仅作演示，省略剩余 Metadata
  "metadata": {
    "SRCL-9520": {
      "title": "僕は存在していなかった",
      "artist": "22/7",
      "tracks": [
        { "title": "僕は存在していなかった" },
        { "title": "地下鉄抵抗主義" },
        { "title": "11人が集まった理由" }
      ]
    }
  },
  // 存放 Annil 的分享 JWT
  // 如果导出的目的并非分享，可以省略
  "tokens": {
    // key 为 annil 的地址，value 为 JWT 内容
    "https://annil.mmf.moe": "a jwt here"
  }
}
```
