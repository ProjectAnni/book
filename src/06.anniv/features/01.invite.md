# 邀请（invite）

## 适用范围

用户注册。

## 参数列表

| 参数名        | 类型     | 详情   |
| ------------- | -------- | ------ |
| `invite_code` | `String` | 邀请码 |

## 错误列表

| 错误代码 | 状态码             | 详情                   |
| -------- | ------------------ | ---------------------- |
| `201000` | `400 Bad Request`  | 邀请系统未开启         |
| `201001` | `403 Forbidden`    | 邀请码无效             |
| `201002` | `401 Unauthorized` | 邀请码与被邀请用户不符 |
| `201003` | `403 Forbidden`    | 邀请码已到达使用上限   |
