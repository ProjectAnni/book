# 邀请（invite）

## 适用范围

- 用户注册

## 请求参数

| 参数名        | 类型     | 详情   |
| ------------- | -------- | ------ |
| `invite_code` | `string` | 邀请码 |

## 错误列表

| 错误代码 | 详情                   |
| -------- | ---------------------- |
| `201000` | 邀请系统未开启         |
| `201001` | 邀请码无效             |
| `201002` | 邀请码与被邀请用户不符 |
| `201003` | 邀请码已到达使用上限   |
