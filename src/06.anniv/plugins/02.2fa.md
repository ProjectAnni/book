# 两步验证（2fa）

## 适用范围

注册、登录等敏感操作处。

## 参数列表

| 参数名     | 类型    | 详情           |
| ---------- | ------- | -------------- |
| `2fa_code` | `\d{6}` | 两步验证的代码 |

## 错误列表

| 错误代码 | 状态码                     | 详情                     |
| -------- | -------------------------- | ------------------------ |
| `202001` | `403 Forbidden`            | 两步验证代码错误         |
| `202002` | `422 Unprocessable Entity` | 用户或服务端未启用 `2FA` |
